## рҹҡҖ Core Features

### рҹӨ– AI-Powered Predictive Matching Engine
Our ML model doesn't wait for a crisis вҖ” it **predicts blood and platelet shortages** up to 72 hours in advance using hospital consumption patterns, local event data, seasonal trends, and historical demand signals. Shortages are prevented, not reacted to.

### вҡЎ Real-Time Compatibility Scoring
Multi-parameter matching across blood type, HLA markers, geographical proximity, donor health history, and recipient urgency level вҖ” returning ranked compatible donors **in under 3 seconds**.

### рҹ“Ҡ Live Shortage Intelligence Dashboard
A public-facing, real-time heatmap of donation demand and supply across Indian cities вҖ” giving hospitals, NGOs, and health authorities actionable intelligence to mobilize faster.

### рҹҚј Human Milk Bank Network (India's First at Scale)
A dedicated sub-platform connecting lactating donors with verified NICU units and premature babies in need. Includes cold chain logistics tracking, pasteurization compliance records, and nutritional profiling вҖ” giving every premature baby a fighting chance.

### рҹ”’ Verified Donor Identity & Health Records
Blockchain-anchored donor health credentials with Aadhaar-linked verification ensure every donor record is authentic, tamper-proof, and instantly accessible to authorized medical staff.

### рҹ“ұ Multilingual Mobile-First Access
Accessible in 10+ Indian languages. Works on low-bandwidth connections. USSD fallback for feature phones вҖ” because the next donor might not have a smartphone.

## рҹ“Ұ Getting Started

### Prerequisites
- Node.js v18+
- Python 3.10+
- PostgreSQL 14+
- Docker & Docker Compose

### Installation

```bash
# Clone the repository
git clone https://github.com/your-org/lifeforge-connect.git
cd lifeforge-connect

# Install frontend dependencies
cd client && npm install

# Install backend dependencies
cd ../server && pip install -r requirements.txt

# Set up environment variables
cp .env.example .env
# Fill in your API keys (Google Maps, Twilio, Aadhaar sandbox, etc.)

# Start all services
docker-compose up --build

# Run ML model setup
cd ml && python setup_models.py
```

### Running in Development

```bash
# Terminal 1 вҖ” Frontend
cd client && npm run dev

# Terminal 2 вҖ” Backend API
cd server && uvicorn main:app --reload

# Terminal 3 вҖ” ML Prediction Service
cd ml && python shortage_predictor.py
```

Visit `http://localhost:3000` to see the platform live.
